# Day 8 - Python Data Logic Sprint Heavy Version

## Business Scenario

This project simulates a real-world order data analysis pipeline for an online business.

The goal is to process raw order records, clean invalid data, validate orders, calculate important business metrics, and generate reports.

The pipeline helps the business understand:
- How many valid orders exist.
- How much revenue was generated from completed orders.
- Which categories and cities perform best.
- Which orders have the highest value.

The project focuses on applying Python data processing logic similar to what is used in real data engineering workflows.
