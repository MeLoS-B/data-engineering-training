

# Day 4 - Python and SQL Logic Review

## Goal of the practice

The goal of this practice was to understand how the same data analysis logic can be solved using both Python and SQL. I practiced filtering data, calculating values, sorting orders, and analyzing business information.

## Files included

- python_orders.py - Python solutions for order analysis tasks.
- setup.sql - SQL table creation and sample order data.
- sql_queries.sql - SQL queries for analyzing the orders.
- comparison.md - Comparison between Python logic and SQL logic.
- daily_report.txt - Short report about the practice.

## How to run python_orders.py

1. Make sure Python is installed.
2. Open the project folder in the terminal.
3. Run:

python python_orders.py
